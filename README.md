DDQ Git Group
=============

The Git Group containing all related DDQ repositories. Uses [gg-core](https://github.com/fidian/gg-core) to group the repositories.

Usage
-----

    # Clone the manifest.
    gg init git@github.com:tests-always-included/ddq-manifest.git

    # Go into the directory.
    cd ddq-manifest

    # Run sync to clone and pull on all repositories.
    gg sync
